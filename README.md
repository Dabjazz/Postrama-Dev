# Postrama-Dev
Application Development related to the postrama social platform. Mainly related to the Android platform might extend to cross-platform.
Postrama, being a social media platform, is still in its concept and mvp phase.
As of 2026 Feb, the MVP is under active development with FOSS software like react-native library, and the use of lovable is heavily involved to develop POCs.
A pure android native implementation is under discussion and will soon be in the development phase.
